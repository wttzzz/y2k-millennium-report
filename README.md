# QQ空间 · 葬爱千禧纪念馆

一个千禧年代、QQ空间、非主流与早期中文互联网文化主题的单页网站。

## 本地打开

推荐用本地服务打开：

```bash
python -m http.server 8787
```

然后访问：

```text
http://127.0.0.1:8787/y2k_millennium_report_music.html
```

## 音乐文件

播放器会读取：

- `assets/music/` 里的音频文件
- `assets/lyrics/` 里的同名 `.lrc` 歌词文件

音频和完整歌词通常受版权保护，仓库不会包含这些文件。请只在本地放入你拥有合法授权的文件。
